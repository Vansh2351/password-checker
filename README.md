# Password Checker

A simple Python script to check the strength of a password based on length, use of uppercase/lowercase letters, numbers, and special characters.

## Features

- Checks if the password meets minimum length requirements
- Validates presence of uppercase and lowercase letters
- Checks for numbers and special characters
- Provides feedback on password strength

## How to Use

1. Make sure you have Python installed (Python 3.x recommended).
2. Clone this repository or download the files.
3. Run the script using the command:

```bash
python password_checker.py
